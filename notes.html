Test feed using `entr`

When you save a ruby file in that directory, the test refreshes and re-runs.

  • Ruby: Minitest

  • ls *.rb | entr ruby *_test.rb

  • Ruby: RSpec

  • ls lib/*.rb | entr rspec spec/*

  • Elixir: Mix

  • ls *exs | entr mix test

  • JavaScript: Yarn

  • ls *.js | entr yarn test
